# Bin_to_header

Saving a binary file in an array of a file with the specified type and extension ".h".  
The resulting file can be linked to main for later use.  
  
Using:  
    bin2header readfile.bin  
Result:  
readfile_bin.bin  
or  
bin2header readfile.bin writefile.h  
Result:  
   writefile.h
